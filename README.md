# MKP Coconut Shop

Welcome to the MKP Coconut Shop project source code.

## How to Run
Simply open the `index.html` file in any modern web browser. 

## Features
- **Product List**: Fresh Coconut, Water, Dry Coconut, Coir.
- **Cart System**: Add items, view summary.
- **Checkout**: Address form and payment selection (GPay, PhonePe, Cash).
- **Admin**: Click "Admin: View Sales" at the bottom to see stored orders.
- **Data Storage**: Orders are saved in your browser's Local Storage.

## Images
Images are located in the `images` folder and were generated for this project.
